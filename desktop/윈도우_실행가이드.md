# 🪟 윈도우(Windows)에서 실행하기 — 쉬운 가이드

> 이 프로그램은 **79대포 지점 네이버 사진을 자동으로 올려주는 앱**이에요.
> 컴퓨터 잘 몰라도 괜찮아요. 한 줄씩 그대로 따라 하세요.
> 빨간 글씨(에러)가 나오면 **화면 사진 찍어서** 도와주는 사람에게 보내세요. 😊

---

## 1. 파이썬(Python) 설치 — 딱 한 번
1. 인터넷에서 **`python.org/downloads`** 접속
2. 노란 **"Download Python"** 버튼 클릭 → 받은 파일 더블클릭
3. ⚠️ **설치 첫 화면에서 맨 아래 "Add Python to PATH" 체크박스를 꼭 켜세요!** (중요)
4. **"Install Now"** 클릭 → 설치 완료

---

## 2. 압축 풀기
- 받은 **`smartplace-cloud.zip`** 마우스 오른쪽 → **"압축 풀기"**
- `smartplace-cloud` 폴더가 생겨요. 그 안의 **`desktop`** 폴더로 들어가기

---

## 3. 실행 — 더블클릭 한 번
- `desktop` 폴더 안의 **`run.bat`** 더블클릭
  - "Windows의 PC 보호" 경고가 뜨면 → **"추가 정보"** → **"실행"** 클릭
- 검은 창이 뜨고 **처음엔 자동 설치(5~10분)** → 끝나면 **앱 창이 떠요**
  > 이 검은 창은 **끄지 마세요!** (끄면 앱도 꺼져요)

---

## 4. 앱에서 사용하기
1. **네이버 로그인** 버튼 → 로그인 창에서 직접 로그인 → 잠시 뒤 **"로그인됨"** ✅
2. **지점 불러오기** (브랜드 번호 `6707`) → 지점 목록이 쭉 떠요
3. **이미지 폴더 선택** → 올릴 사진들이 든 폴더
4. **등록 실행** → 한 곳씩 ✅ 표시되며 등록돼요

> 다음부터는 **`run.bat` 더블클릭** 한 번이면 바로 앱이 켜져요.

---

## ❓ 자주 막히는 곳
| 증상 | 해결 |
|---|---|
| `python` 이 없다는 에러 | 1단계에서 **"Add Python to PATH" 체크** 하고 다시 설치 |
| "Windows의 PC 보호" 경고 | "추가 정보" → "실행" (이 앱은 안전해요) |
| 로그인/등록 안 됨 | 네이버 로그인부터 다시 |
| 빨간 글씨 에러 | 화면 **사진 찍어서** 도와주는 사람에게 |

**혼자 끙끙대지 말고 캡처해서 물어보세요 — 그게 제일 빨라요!** 🙂
