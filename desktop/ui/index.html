3. 작업 선택

① 양식 받기 → 엑셀로 메뉴 채우기 → ② 작성한 파일 선택

선택한 지점의 방문·조회·리뷰·예약수를 모아 CSV로 저장합니다. 읽기 전용이라 매장 정보는 바뀌지 않아요. (베타: 읽지 못한 지표는 빈 칸으로 표시)

선택한 지점의 리뷰(작성자·별점·내용·작성일)를 한곳에 모아 CSV로 저장합니다. 읽기 전용이라 매장 정보는 바뀌지 않아요. (베타: 읽지 못한 지점은 '읽기 실패'로 표시)

AI 답글 초안 (선택) — Anthropic API 키 필요

수집한 리뷰에 사장님 답글 초안을 Claude로 생성합니다. 초안만 만들고, 검토 후 직접 게시하세요. 리뷰 본문이 Anthropic으로 전송됩니다.
