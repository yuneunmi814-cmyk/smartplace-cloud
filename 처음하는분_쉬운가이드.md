# 🟢 처음 하는 분을 위한 쉬운 가이드

> 이 프로그램은 **79대포 지점들 네이버에 사진을 자동으로 올려주는** 도구예요.
> 컴퓨터 잘 몰라도 괜찮아요. **위에서부터 한 줄씩 그대로** 따라 하면 돼요.
> 막히면 그 화면을 **사진 찍어서** 도와주는 사람에게 물어보세요. 😊

이 가이드는 맥북(Mac) 기준이에요.

---

## 준비물 (먼저 받을 것)
도와주는 사람(넘긴 사람)에게 **이 3가지**를 미리 받으세요:
1. **네이버 아이디 / 비밀번호** (79대포 본사 계정)
2. (사진 올릴) **이미지들** — 한 폴더에 모아두기
3. 이 프로그램 압축파일 (`smartplace-cloud.zip`)

---

## 1단계 · 파이썬(Python) 설치
프로그램이 돌아가려면 '파이썬'이 필요해요. 한 번만 설치하면 돼요.

1. 인터넷에서 **python.org/downloads** 접속
2. 노란 버튼 **"Download Python"** 클릭 → 다운로드된 파일 더블클릭
3. 창이 뜨면 **"계속 / 다음 / 동의"** 만 계속 누르기 → 설치 완료

---

## 2단계 · 프로그램 설치 (자동)
1. 받은 **`smartplace-cloud.zip`** 더블클릭해서 압축 풀기
   → `smartplace-cloud` 폴더가 생겨요 (어디 있는지 기억! 보통 '다운로드' 폴더)

2. **터미널** 열기:
   - 키보드 `⌘(커맨드) + 스페이스` 누르기 → `터미널` 입력 → Enter
   - 까만(또는 흰) 창이 떠요. 여기에 글자를 붙여넣을 거예요.

3. 터미널에 이렇게 **딱 이 글자만** 입력하고 (띄어쓰기 포함) 한 칸 띄우기:
   ```
   bash 
   ```
   (`bash` 치고 **스페이스 한 번**)

4. 그 다음 **`smartplace-cloud` 폴더 안의 `setup.sh` 파일을** 터미널 창으로 **드래그**해서 놓기
   → 자동으로 경로가 채워져요. 그리고 **Enter**.

5. 설치가 시작돼요 (몇 분 걸림). **`✅ 설치 완료!`** 가 뜰 때까지 기다리세요.

---

## 3단계 · 네이버 로그인 (딱 한 번)
터미널에 아래를 **한 줄씩** 붙여넣고 each Enter:

```
cd ~/Downloads/smartplace-cloud/gateway
```
```
source .venv/bin/activate
```
```
python -m app.seed_session 79daepo
```

→ **브라우저 창이 자동으로 떠요.** 거기서 **네이버에 직접 로그인**하세요
(아이디·비밀번호 입력, 문자인증 같은 거 나오면 그것도 직접).
로그인 후 **스마트플레이스 화면까지** 들어가면, 터미널에 **`✅ 세션 저장 완료`** 가 떠요.

> 폴더가 '다운로드'가 아니면 `~/Downloads/` 부분만 실제 위치로 바꾸세요.

---

## 4단계 · 지점 목록 가져오기
터미널에 붙여넣고 Enter:
```
python -m app.inspect 79daepo brand 6707
```
→ `placeSeq 추출: 132곳` 처럼 지점이 나오면 성공.

---

## 5단계 · 사진 올리기 (자동!)
준비한 **이미지 폴더**를 사진 올릴 곳에 적용해요.

```
GATEWAY_MOCK=0 python -m app.bulk_brand 79daepo 6707 
```
위 글자 마지막에 **스페이스 한 칸** 띄우고 → **이미지 폴더를 터미널로 드래그** → Enter.

- `계속하려면 Enter` 가 뜨면 → **Enter**
- 지점이 하나씩 `✅` 표시되며 사진이 올라가요 (개수 많으면 시간 걸림)
- 다 끝나면 **성공/실패 요약**이 나와요

> 처음엔 전체 말고 **몇 곳만 테스트**가 안전해요:
> 위 명령 맨 뒤에 ` --limit 2` 를 붙이면 2곳만 해봐요.

---

## ❓ 자주 막히는 곳
| 증상 | 해결 |
|---|---|
| `command not found: python` | 1단계 파이썬 설치 다시 |
| 브라우저 로그인 후 멈춤 | 스마트플레이스(`new.smartplace.naver.com`)까지 들어갔는지 확인 |
| `세션이 없습니다` | 3단계 네이버 로그인 다시 |
| 빨간 글씨 에러 | 그 화면 **사진 찍어서** 도와주는 사람에게 |

**막히면 절대 혼자 끙끙대지 말고, 화면 캡처해서 물어보세요!** 그게 제일 빨라요 🙂
