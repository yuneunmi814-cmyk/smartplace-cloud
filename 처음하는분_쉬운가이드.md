# 🟢 처음 하는 분을 위한 쉬운 가이드 (웹앱 버전)

> 이 프로그램은 **79대포 지점 네이버 사진을 올려주는 관리 사이트**예요.
> 컴퓨터 잘 몰라도 괜찮아요. **위에서부터 한 줄씩 그대로** 따라 하세요.
> 빨간 글씨(에러)가 나오면 **그 화면을 사진 찍어** 도와주는 사람에게 보내세요. 😊

맥북(Mac) 기준이에요.

---

## 0. 먼저 받을 것 (도와주는 사람에게)
1. 프로그램 압축파일 **`smartplace-cloud.zip`**
2. **`.env` 라는 작은 파일** (← 비밀 설정. 메일·메신저로 받기)
3. **네이버 아이디 / 비밀번호** (79대포 계정)

---

## 1. 프로그램 2개 설치 (각각 한 번만)
사이트가 돌아가려면 'Python'과 'Node' 두 개가 필요해요.

- **Python**: 인터넷에 `python.org/downloads` → 노란 **Download** 버튼 → 받은 파일 더블클릭 → "계속/동의"만 누르기
- **Node**: 인터넷에 `nodejs.org` → 초록 **LTS** 버튼 → 받은 파일 더블클릭 → "계속/동의"만 누르기

---

## 2. 압축 풀고 → 설치 실행
1. **`smartplace-cloud.zip`** 더블클릭 → `smartplace-cloud` 폴더 생김 (보통 '다운로드'에)
2. **받은 `.env` 파일을 → `smartplace-cloud` 안의 `backend` 폴더 안에** 넣기 (그냥 드래그해서 복사)
3. **터미널** 열기: `⌘ + 스페이스` → `터미널` 입력 → Enter
4. 터미널에 `bash ` 치고 **스페이스 한 칸** → `smartplace-cloud` 폴더 안의 **`setup.sh` 파일을 드래그** → Enter
5. **`✅ 설치 완료!`** 가 뜰 때까지 기다리기 (5~10분)

---

## 3. 네이버 로그인 (딱 한 번)
터미널에 **한 줄씩** 붙여넣고 each Enter:
```
cd ~/Downloads/smartplace-cloud/gateway
```
```
source .venv/bin/activate
```
```
python -m app.seed_session 79daepo
```
→ 브라우저가 뜨면 **네이버에 직접 로그인** → 스마트플레이스 화면까지 들어가면 **`✅ 세션 저장 완료`**.

---

## 4. 사이트 켜기 🚀
새 터미널을 열거나 같은 창에서:
```
cd ~/Downloads/smartplace-cloud
```
```
bash start.sh
```
→ 10초 뒤 **브라우저에 사이트가 자동으로 열려요** (`localhost:5173`).
> 이 검은 창은 **끄지 마세요!** (끄면 사이트도 꺼져요) 다 쓰고 끌 땐 이 창에서 `Control + C`.

---

## 5. 사이트 사용하기
1. **회원가입** (이메일·비밀번호 아무거나) → 첫 가입자가 관리자예요
2. **계정·가맹점** 탭 → 네이버 계정 연동 (아이디에 **`79daepo`** 똑같이 입력)
3. **가맹점 등록** → 관리할 지점 추가
   - (지점 많으면: 새 터미널에서 `cd ~/Downloads/smartplace-cloud/backend && source .venv/bin/activate && python -m scripts.seed_naver 79daepo` 한 줄이면 40곳 자동 등록)
4. **이미지** 탭 → 사진 업로드 (여러 장 한 번에 OK)
5. **배포** 탭 → 이미지·지점 골라서 **배포 실행**
6. **작업 현황** 탭 → 진행 상황 보기

---

## ❓ 자주 막히는 곳
| 증상 | 해결 |
|---|---|
| `command not found` | 1단계 Python/Node 설치 다시 |
| 사이트가 안 열림 | `start.sh` 창이 켜져 있는지 확인, 잠시 더 기다리기 |
| 로그인/배포 안 됨 | 3단계 네이버 로그인(`seed_session`) 다시 |
| 빨간 글씨 에러 | 화면 **사진 찍어서** 도와주는 사람에게 |

**혼자 끙끙대지 말고 화면 캡처해서 물어보세요 — 그게 제일 빨라요!** 🙂
